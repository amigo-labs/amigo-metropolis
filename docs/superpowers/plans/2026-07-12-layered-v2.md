> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deterministische, begehbare Mehrdeck-Laufflächen (bis zu N=3 Ebenen über einer (x,y)) im Sim, plus Tier-1-Rendering der oberen Decks — validiert gegen eine synthetische Testmap.

**Architecture:** N gestapelte Heightfields in `MapData` (Layer 0 = bestehendes `heights`, obere Ebenen additiv + leer bei einstöckigen Maps). Per-Entity-`layer` in einem **Seiten-Array** `SimState.entLayer`, das **nur für layered Maps** gehasht wird → einstöckige Maps bleiben byte-identisch (No-op-Invariante wie bei den Wänden). Zentrale `resolveHeight(map,x,y,layer)`; Walker/Bodeneinheiten wählen pro Tick die höchste erreichbare Fläche (`≤ h + STEP_SNAP`), sonst Gravity-Fall. Separation nur innerhalb derselben Ebene.

**Tech Stack:** TypeScript (strict), Bun (Tests via `bun test`, `bun:test`), Biome (lint/format), Three.js (Client-Render). Keine Runtime-Deps in `packages/sim`.

**Spec:** `docs/superpowers/specs/2026-07-12-layered-v2-design.md`

## Global Constraints

- **Determinismus (`packages/sim/src`):** nur `+ - * /`, `Math.sqrt/floor/ceil/abs/min/max/sign`, Integer-Bitwise. KEIN `sin/cos/tan/atan2/pow/exp/log/random` (LUT in `simMath.ts` nutzen), KEIN `Date.now`/`performance.now`/`new Date`, KEIN `async`/`await`/`Promise`, KEINE DOM-Globals. `determinismGuard.test.ts` scannt statisch — muss grün bleiben.
- **No-op-Invariante (tragend):** Einstöckige Maps (`map.layerHeights.length === 0`) müssen byte-identische Hash-Sequenzen behalten. Jede neue Höhen-/Layer-Logik early-outet als ERSTE Anweisung auf leere Layer-Arrays. Bewiesen dadurch, dass `golden.test.ts` OHNE Golden-Regen grün bleibt, bis Task 7 explizit `SIM_VERSION` bumpt.
- **Iterationsreihenfolge:** immer `for id in [0, high)` und `!alive` überspringen — nie über `Map`/`Set`/Objekt-Keys.
- **`spawn`/`despawn` (`entities.ts`) müssen jedes gehashte Feld setzen/nullen** — ein freed slot muss identisch zu einem nie-genutzten hashen.
- **SIM_VERSION-Bump = Golden-Regen im selben Commit** (`bun tools/replay/src/cli.ts record …`), mit Begründung in der Commit-Message. No-op-Beweis: Hash-Arrays der einstöckigen Goldens elementweise gegen `git show HEAD:` vergleichen.
- **`packages/sim/package.json` darf keine `dependencies` haben** (Null-Runtime-Deps; vom Guard geprüft).
- **Commits:** Conventional Commits, imperativ, englisch. Co-Authored-By-Trailer wie im Repo üblich.
- **Scope-Grenze:** NUR Fortbewegung ist layered. Kampf/LOS/Wände bleiben 2D (layer-agnostisch). Der private FCOP-Extractor + `tools/fcop/convert.ts`-Erweiterung sind **Sub-Projekt 3** und NICHT Teil dieses Plans.

**Verifikation nach jeder Task:** `bun test` grün, `bun run lint` grün, `bun run typecheck` grün. Nach Task 7 zusätzlich `bun run replay:verify`.

```ts
/**
 * Bilinear height sample on an EXTRA layer (layerIdx 0-based into layerHeights,
 * i.e. layerIdx 0 == layer 1). Same math as sampleHeight — bit-exact on every
 * engine. Callers guard layerIdx in range; out-of-range degrades to base.
 */
export function sampleLayerHeight(map: MapData, layerIdx: number, x: number, y: number): number {
  if (layerIdx < 0 || layerIdx >= map.layerHeights.length) return sampleHeight(map, x, y);
  const heights = map.layerHeights[layerIdx];
  const max = map.size - 1;
  const gx = clamp(x / map.cellSize, 0, max);
  const gy = clamp(y / map.cellSize, 0, max);
  let i0 = Math.floor(gx);
  let j0 = Math.floor(gy);
  if (i0 > max - 1) i0 = max - 1;
  if (j0 > max - 1) j0 = max - 1;
  const fx = gx - i0;
  const fy = gy - j0;
  const s = map.size;
  const row0 = j0 * s + i0;
  const row1 = row0 + s;
  const h0 = lerp(heights[row0], heights[row0 + 1], fx);
  const h1 = lerp(heights[row1], heights[row1 + 1], fx);
  return lerp(h0, h1, fy);
}

/**
 * Height of the walkable surface at (x, y) on `layer` (0 = base heights).
 * The layer===0 / no-extra-layers early-out makes single-story maps bit-
 * identical to a plain sampleHeight (No-op invariant).
 */
export function resolveHeight(map: MapData, x: number, y: number, layer: number): number {
  if (layer === 0 || map.layerHeights.length === 0) return sampleHeight(map, x, y);
  return sampleLayerHeight(map, layer - 1, x, y);
}
```

```ts
/**
 * Walker/ground layer resolution: among the base surface and every extra deck
 * present at (x, y), pick the HIGHEST whose surface is within STEP_SNAP above
 * the current height h (i.e. steppable). Anything higher is an overhang you
 * walk under; if only a far-below surface qualifies you are airborne and fall.
 * On single-story maps this returns { layer: 0, height: sampleHeight } —
 * bit-identical to the old rideHeight(walker) path (No-op invariant).
 */
function walkerRide(state: SimState, x: number, y: number, h: number): { layer: number; height: number } {
  const map = state.map;
  const baseH = sampleHeight(map, x, y);
  if (map.layerHeights.length === 0) return { layer: 0, height: baseH };
  const s = map.size;
  const inv = 1 / map.cellSize;
  let i = Math.floor(x * inv);
  let j = Math.floor(y * inv);
  if (i < 0) i = 0;
  else if (i > s - 1) i = s - 1;
  if (j < 0) j = 0;
  else if (j > s - 1) j = s - 1;
  let bestLayer = 0;
  let bestH = baseH;
  for (let L = 0; L < map.layerHeights.length; L++) {
    if (map.layerMask[L][j * s + i] === 0) continue;
    const hs = sampleLayerHeight(map, L, x, y);
    if (hs <= h + STEP_SNAP && hs > bestH) {
      bestLayer = L + 1;
      bestH = hs;
    }
  }
  return { layer: bestLayer, height: bestH };
}
```

- [ ] **Step 1: Implementieren** — in `terrain.ts` (Muster wie `buildTerrainMesh`, aber nur Quads, deren 4 Ecken alle in `layerMask` präsent sind):

```ts
/**
 * Tier-1 upper-deck render: one vertex-colored mesh per extra layer, built from
 * the SAME layerHeights/layerMask the collision reads (single source of truth).
 * Only quads whose four corners are all present in the mask are emitted, so a
 * deck renders exactly where it is walkable. Empty for single-story maps.
 */
export function buildDeckMeshes(map: MapData): THREE.Mesh[] {
  const meshes: THREE.Mesh[] = [];
  const s = map.size;
  const cell = map.cellSize;
  for (let L = 0; L < map.layerHeights.length; L++) {
    const heights = map.layerHeights[L];
    const mask = map.layerMask[L];
    const positions: number[] = [];
    const indices: number[] = [];
    const present = (i: number, j: number): boolean => mask[j * s + i] === 1;
    for (let j = 0; j < s - 1; j++) {
      for (let i = 0; i < s - 1; i++) {
        if (!present(i, j) || !present(i + 1, j) || !present(i, j + 1) || !present(i + 1, j + 1)) {
          continue;
        }
        const base = positions.length / 3;
        const push = (ci: number, cj: number): void => {
          positions.push(ci * cell, heights[cj * s + ci], cj * cell);
        };
        push(i, j);
        push(i + 1, j);
        push(i, j + 1);
        push(i + 1, j + 1);
        indices.push(base, base + 2, base + 1, base + 1, base + 2, base + 3);
      }
    }
    if (indices.length === 0) continue;
    const geometry = new THREE.BufferGeometry();
    geometry.setAttribute("position", new THREE.BufferAttribute(new Float32Array(positions), 3));
    geometry.setIndex(new THREE.BufferAttribute(new Uint32Array(indices), 1));
    geometry.computeVertexNormals();
    const material = new THREE.MeshStandardMaterial({
      color: HIGH_COLOR.clone().multiplyScalar(0.85),
      flatShading: true,
      side: THREE.DoubleSide,
    });
    const mesh = new THREE.Mesh(geometry, material);
    mesh.matrixAutoUpdate = false;
    mesh.updateMatrix();
    meshes.push(mesh);
  }
  return meshes;
}
```
